# Feature Specification: Monthly Budget

**Feature Branch**: `004-monthly-budget`
**Created**: 2026-04-23
**Status**: Draft

## Context

Users want to set spending limits for each category and an
overall monthly budget so they can track their spending
against their goals. Budgets reset on the 1st of every
month automatically.

## User Scenarios & Testing

### User Story 1 - Set a Monthly Budget (Priority: P1)

A user wants to set an overall monthly budget and per
category budgets so they have spending targets to work
towards.

**Why this priority**: Core feature — without setting a
budget nothing else in this feature works.
